# Learn_Geometric_Shapes.py

Geometric Shapes Learning App
An interactive Python application for children to learn geometric shapes.
About
This educational app helps kids learn basic geometric shapes through colorful buttons and visual representations. Click on a shape name to see its graphic.
Features

8 different shapes: Circle, Square, Triangle, Rectangle, Star, Hexagon, Heart, Oval
Colorful and kid-friendly interface
Interactive buttons
English language
